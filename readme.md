# RFUNCS

A collection of amazing functions:

## all

_all(array)_

-   Returns `true` if all values in the array are true

```javascript
all()
// => true
```

```javascript
all(true)
// => true
```

```javascript
all(false)
// => false
```

```javascript
all([true, true])
// => true
```

```javascript
all([true, false])
// => false
```

_all(array, func)_

-   Returns `true` if all values in the array are true when applied to func

```javascript
all(4, i => i <= 4)
// => true
```

```javascript
all(4, i => i > 4)
// => false
```

```javascript
all([2, 4], i => i <= 2)
// => false
```

```javascript
all([2, 4], i => i <= 4)
// => true
```

```javascript

// => false
```



## apply

_apply(f, [val₁, val₂, val₃, …])_

-   returns f(val₁, val₂, val₃, …)

```javascript
apply(a => a + 2, 1)
// => 3
```

```javascript
apply(a => a + 2, [1])
// => 3
```

```javascript
apply((a, b) => a + b, [1, 2])
// => 3
```

```javascript
apply((a, b, c) => a + b + c, [1, 2, 3])
// => 6
```

## composition

_composition(f₁, f₂, f₃, …)_

-   returns a composition of the functions f₁, f₂, f₃, …

```javascript
composition()(4)
// => 4
```

```javascript
composition(a => a + 2)(4)
// => 6
```

```javascript
composition(
    a => a + 2,
    a => a * 3
)(4)
// => 18
```

```javascript
composition(
    a => a + 2,
    a => a * 3,
    a => a / 2
)(4)
// => 9
```

```javascript
// => [1]
```

```javascript
// => []
```

